Talko

Type:
Personal UI/UX Project

Duration:
03-05.2021

Talko is a portable public speaking game app designed as a solution to public speaking problems of individuals.

It is designed as an interactive game which user can access from mobile platforms. Taking inspiration from Just Dance, a rhythm game series developed and published by Ubisoft, Talko allows its user practice public speaking on Televisions through Airplay. To better evaluate the performance of speaker, a gesture capturing glove is designed to detect the motion when speaking.

Specification

User research, User flow, Information architecture, Wireframe, Icon, Hi-fidelity interface
